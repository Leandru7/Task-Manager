# Task Manager

Gerenciador de tarefas moderno e responsivo construído com HTML, CSS e JavaScript vanilla.

## 🚀 Funcionalidades
- Adicionar, editar e remover tarefas
- Sistema de prioridades
- Filtros inteligentes
- Persistência local
- Design responsivo

## 🛠️ Tecnologias
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript ES6+
- LocalStorage API
